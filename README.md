## 使用套件安裝: 
    $ pip install django 
    $ pip install djangorestframework
    $ pip install django-crispy-forms

## 安裝方式
    $ git clone https://github.com/Evan-codetest/Short_url.git
    $ cd project_url
    
    $ python manage.py migrate
    $ python manage.py runserver
    
    # 使用瀏覽器打開 http://127.0.0.1:8000/
